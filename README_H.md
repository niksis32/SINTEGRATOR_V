# Синтегратор — `sintegrator-site_v`

Одностраничный промо-сайт на **React 19 + Vite 7**: светлый промышленный стиль («machined steel») по эталону из **`stitch_sintegrator_enterprise_digital_architecture/`**, шрифты Space Grotesk + Inter, иконки Material Symbols.

## Быстрый старт

```bash
cd sintegrator-site_v
npm install
npm run dev
```

Откройте `http://localhost:5173` (или другой порт через `-- --port …`).

## Сборка

```bash
npm run build
```

Артефакты — в **`dist/`** (выкладывать на хостинг как статику).

---

**Подробно:** что сделано, перенос в другую папку, PowerShell/npm, продакшен и правки контента — в файле **`ИНСТРУКЦИЯ-РАЗВЕРТЫВАНИЯ.md`**.
